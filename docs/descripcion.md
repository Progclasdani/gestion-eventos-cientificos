# Descripción del programa.
## Resumen breve de funcionalidad
Este programa establece las funcionalidades básicas para el manejo de datos científicos. El prototipo dispondrá de las 
funcionalidades de un menú de opciones a seleccionar, entrada de datos de puntos espaciales y la cración de un diagrama
que muestre los puntos almacenados.
## Opciones de menú
Se dispone de las siguientes opciones en el menú que se pueden seleccionar meniante la introducción del número de opción:
1. Añadir puntos
2. Mostrar diagrama de puntos
3. Salir del programa
### Añadir puntos
Esta opción permite abrir una interación con el usuario para tomar las coordenas de los puntos en "x" e "y". Estos valores
son almacenados por el programa para poder pasarlos a la función de generación de la gráfica.
